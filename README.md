# Učimo Arduino

Učimo Arduino i programiranje mikrokontrolera

## Razvoj

Po povezivanju Arduina, podesiti Arduino IDE -> Tools -> Port

Za ispis u konzolu otvoriti: Arduino IDE -> Tools -> Serial Monitor
